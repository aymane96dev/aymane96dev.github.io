# Personal Portfolio

Welcome to my personal portfolio repository! This project showcases my skills, projects, and experience in web development.
It is built using modern web technologies to create an interactive and visually appealing online portfolio.

## About

This portfolio is a comprehensive showcase of my professional journey, including:
- A brief introduction about myself.
- An overview of my skills and technologies I am proficient in.
- Detailed descriptions of my projects with links to live demos and source code.

## Features

- **Responsive Design**: Looks great on both desktop and mobile devices.
- **Interactive UI**: Smooth scrolling, hover effects, and animations.
- **Projects Gallery**: Filterable project gallery with detailed modals.

## Technologies Used

- **Frontend**:
  - HTML5
  - CSS3
  - JavaScript (ES6+)
